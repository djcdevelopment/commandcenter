The Two Economies & the Wind Tunnel

The laboratory runs on two economies. Economy 1 — metered compute: every frontier API call spends dollars, rate limit, and availability. Economy 2 — sunk compute: the hardware is already purchased; the marginal cost of one more experiment is nearly zero, bounded only by electricity, heat, wear, and Derek's attention. They demand different objective functions — Economy 1 optimizes cost per outcome; Economy 2 optimizes knowledge produced per hour of local runtime. "Laps on electricity," formalized.

Δ3

Learning rate — the Economy-2 objective, derived

DRAFT

What

A projection, per node and per economy: knowledge per hour of runtime = beliefs changed (experiment results already compute belief_before/after) + associations formed + qualifications renewed + coverage gaps closed, over observed runtime hours. The number "run it hot" is actually maximizing.

Why it's honest

Every term already exists as a derived object — the metric is a sum over the spine, never asserted. A hot GPU producing no belief change scores zero: utilization is not the objective, learning is.

Δ4

Idle-drain dispatch — the wind tunnel's engine

DRAFT

What

When a sunk-economy node goes idle, the scheduler pulls the highest-value experiment candidate and runs it as a normal gated dispatch: plan authored, evidence observed, belief re-judged, capability updated. The tunnel spins whenever Derek isn't using the machine — and stops the moment he is.

What this actually is

Stage-6 autonomy arriving through economics, not through ambition. No new objects: candidates → plans → results → beliefs all shipped. The only new decision is "the queue may self-serve when the marginal cost is zero" — which is an authored policy, on the record, suspendable like any override.

Δ5

Mission replay — the race engineer's station

VIEW

What

Select a completed run; scrub a timeline: accepted → planned → policy matched → assigned → model warm → GPU ramp → fans → context → assay → promotion → finding updated. Events give the skeleton, decisions give the reasoning, observations give the outcome, Δ2 gives the physiology overlay.

The questions it answers

Why did this mission slow — policy or memory limit? Did the planner force a reload? Did thermals cut sustained throughput? Hardware bottleneck or organizational bottleneck? A projection answers all of them, because every input is already on the record.

Constitution check

Nothing poured in authors a truth.

Learning rate, thermal envelopes, sustained-decay findings, replay — all projections over events. The wind tunnel adds zero hand-maintained state.

The new authored residue is exactly two objects — and both are will, not belief.

Observe-this declarations (Δ2's telemetry fields — what the organism can feel is Derek's call) and the operating budget: how hot, how loud, how much wear the laboratory may spend when it runs unattended. The budget is a risk-acceptance object, the same species as an experiment plan's risk_accepted — authored, reasoned, on the audit trail, suspendable. The constitution survives contact with the physical world.
